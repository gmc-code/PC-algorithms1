Assignment
--------------------------

| Assignment is an instruction that assigns a value to a variable.

.. code-block:: python

    num = 7


| **Pseudocode**. Assignment uses a left pointing arrow in pseudocode.
| The equivalent pseudocode is:

.. code-block::

    num <- 7
